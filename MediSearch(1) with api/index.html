About MediSearch

The MediSearch is a web-based application designed to assist users in identifying possible diseases based on their symptoms and finding suitable medicines. The platform allows users to enter their symptoms, after which the system searches the database and displays relevant diseases, recommended medications, and additional information.

The website also provides the option to view medicines prescribed by doctors for accurate treatment guidance. This project aims to create an easy-to-use interface backed by a reliable database that stores information about diseases, symptoms, and medicines.

It is particularly useful for preliminary health awareness and as a quick reference tool. While it does not replace professional medical advice, it serves as an educational and supportive platform for users seeking health-related information.
